## 📖 About the Project

**GWA Reveal** is a lightweight and intuitive web-based GPA/GWA calculator built to help students quickly compute their grades with ease.

This project focuses on strengthening **core JavaScript fundamentals**, including DOM manipulation, user input handling, and real-time computation.
