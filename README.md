# GWA Reveal

GWA Reveal is a simple web-based GPA / GWA calculator designed to help students  
quickly compute their grades.  
This project was created as part of JavaScript practice to strengthen core web development skills.

## Sample Image

![Sample Image](https://github.com/Kiks777/gwa-reveal/blob/main/public/gwa-reveal.png)

## Features

- Calculates GPA / GWA based on user input
- Simple and easy-to-use interface
- Designed for quick grade computation
- Built as a practice project for JavaScript fundamentals

## Tech Stack

- **Frontend:** HTML, JavaScript  
- **Styling:** Tailwind CSS  
- **Deployment:** GitHub Pages
